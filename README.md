# admanage
AD管理工具
通过flask结合python的ldap库实现页面化的AD管理
同时对接了北森和163邮箱，实现了自动同步北森架构和人员，自动获取北森待入职人员信息，自动输入AD，自动将AD组织架构同步到163邮箱。
